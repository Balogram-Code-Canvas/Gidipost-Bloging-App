# Gidipost

Gidipost is a full-stack blogging platform for tech lovers 
worldwide. From software development and programming to AI, 
gadgets, tech news, scholarships, and more — Gidipost covers 
it all.

🔗 Live: [gidipost.com](https://gidipost.com)

---

## About

Gidipost is a modern blogging platform built for everyone 
who loves technology. Whether you are a developer, student, 
tech enthusiast, or just someone curious about the world of 
technology — Gidipost has something for you.

We cover a wide range of topics including software 
development, programming languages, artificial intelligence, 
phones and gadgets, tech scholarships, trending tech news, 
and much more.

---

## Features

- Read and explore blog posts across multiple tech categories
- AI-assisted content generation
- Rich text blog editor with image support
- Fast image delivery and optimization
- Comment system on blog posts
- Category filtering and blog search
- Fully responsive on all devices
